# Анализ гипотезы о завышении рейтингов на Fandango

## О проекте
Цель проекта - проверить гипотезу о том, что **Fandango** показывает более высокие рейтинги фильмов, чем другие популярные платформы, и визуально завышает оценки по сравнению с собственной числовой пользовательской оценкой.

Проект оформлен как небольшое исследование с тремя уровнями анализа:
- внутреннее сравнение метрик Fandango;
- контекстное сравнение с Rotten Tomatoes и Metacritic;
- итоговое сопоставление Fandango с Rotten Tomatoes, Metacritic и IMDb на общей шкале.

## Исследовательский вопрос
Есть ли у Fandango систематическое смещение рейтингов вверх относительно:
- собственной пользовательской оценки `RATING`;
- других сайтов с рейтингами фильмов.

## Данные
Используются два набора данных:
- `data/fandango_scrape.csv` - фильм, отображаемые звезды `STARS`, пользовательский рейтинг `RATING`, количество голосов `VOTES`.
- `data/all_sites_scores.csv` - оценки тех же или пересекающихся фильмов на Rotten Tomatoes, Metacritic и IMDb.

## Структура проекта
- `analysis/fandango_analysis.ipynb` - локальный анализ данных Fandango.
- `analysis/all_sites.ipynb` - контекстный анализ расхождений между критиками и пользователями на других площадках.
- `analysis/matching.ipynb` - основное сравнение Fandango с другими сайтами на единой шкале `0..5`.
- `analysis/summary.ipynb` - сводный ноутбук с цельной историей исследования от гипотезы до вывода.
- `scripts/analyze_fandango.py` - воспроизводимый скрипт с расчётом ключевых метрик.

## Ход анализа

### 1. Внутренний анализ Fandango
Проверяется:
- связь между рейтингом и количеством голосов;
- различие между `STARS` и `RATING`;
- распределение величины `STARS - RATING`.

Ключевые метрики:
- фильмов с ненулевым числом голосов: `435`;
- среднее `STARS`: `4.123`;
- среднее `RATING`: `3.911`;
- средняя разница `STARS - RATING`: `0.212`.

Интерпретация: отображаемые звезды на Fandango в среднем выше, чем численный рейтинг пользователей. Это уже указывает на смещение визуальной оценки вверх.

### 2. Контекст по другим платформам
В `all_sites.ipynb` сравниваются оценки критиков и пользователей на Rotten Tomatoes и Metacritic.

Задача этого шага - показать, что различия между аудиториями на агрегаторах нормальны, но они не тождественны эффекту, когда сама площадка отображает итоговую оценку более высокой.

### 3. Сопоставление сайтов на общей шкале
В `matching.ipynb` данные объединяются по общим фильмам и нормализуются к шкале `0..5`.

Средние значения на пересечении датасетов (`145` фильмов):
- `Fandango STARS`: `4.09`;
- `Fandango RATING`: `3.84`;
- `Rotten Tomatoes critics`: `3.03`;
- `Rotten Tomatoes users`: `3.18`;
- `Metacritic critics`: `2.93`;
- `Metacritic users`: `3.25`;
- `IMDb`: `3.36`.

Это показывает, что Fandango в среднем оценивает фильмы заметно выше конкурентов.

### 4. Проверка на слабых фильмах
Для 10 худших фильмов по Rotten Tomatoes:
- `Fandango STARS`: `3.75`;
- `Fandango RATING`: `3.47`;
- `Rotten Tomatoes critics`: `0.47`;
- `Rotten Tomatoes users`: `1.78`;
- `Metacritic critics`: `1.34`;
- `Metacritic users`: `1.76`;
- `IMDb`: `2.62`.

Это особенно сильный аргумент: даже для фильмов, которые другие платформы оценивают очень низко, Fandango остаётся сравнительно высоким.

## Вывод
Гипотеза на используемой выборке подтверждается.

Проект показывает два устойчивых эффекта:
1. Внутри Fandango отображаемые звезды `STARS` в среднем выше пользовательского рейтинга `RATING`.
2. При сравнении с Rotten Tomatoes, Metacritic и IMDb оценки Fandango систематически смещены вверх.

Корректный итоговый тезис:
**Fandango на этом датасете демонстрирует сильные признаки завышения рейтингов по сравнению с другими сайтами и собственной числовой пользовательской оценкой.**

## Как запустить
Установка зависимостей:

```bash
poetry install
```

Запуск ноутбуков:

```bash
poetry run jupyter notebook
```

Запуск воспроизводимого скрипта с ключевыми метриками:

```bash
poetry run python scripts/analyze_fandango.py
```
