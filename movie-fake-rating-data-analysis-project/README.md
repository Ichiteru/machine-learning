# Анализ гипотезы о завышении рейтингов на Fandango

## Цель проекта
Проверить гипотезу о том, что сайт **Fandango** показывает более высокие рейтинги фильмов, чем другие популярные платформы, и визуально завышает оценки по сравнению с внутренним пользовательским рейтингом и внешними агрегаторами.

## Данные
В проекте используются два набора данных:

- `data/fandango_scrape.csv` - данные по фильмам с Fandango: название фильма, отображаемые звезды (`STARS`), численный рейтинг пользователей (`RATING`) и число голосов (`VOTES`).
- `data/all_sites_scores.csv` - оценки тех же или пересекающихся фильмов на Rotten Tomatoes, Metacritic и IMDb.

## Структура анализа

### 1. `analysis/fandango_analysis.ipynb`
Локальный анализ данных Fandango:

- проверяется связь между рейтингом и количеством голосов;
- сравниваются распределения `STARS` и `RATING`;
- анализируется разница `STARS - RATING`;
- проверяется состав выборки по годам.

Ключевое наблюдение: среди фильмов с голосами отображаемые звезды на Fandango в среднем выше численного рейтинга пользователей.

Фактические значения на текущем датасете:

- фильмов с ненулевым числом голосов: `435`;
- среднее `STARS`: `4.123`;
- среднее `RATING`: `3.911`;
- средняя разница `STARS - RATING`: `0.212`.

Это означает, что интерфейс Fandango систематически показывает более высокую визуальную оценку, чем следует из численного рейтинга.

### 2. `analysis/all_sites.ipynb`
Контекстный анализ Rotten Tomatoes и Metacritic:

- сравниваются оценки критиков и пользователей;
- исследуется направление и величина расхождений.

Этот шаг нужен, чтобы отделить обычные расхождения между аудиториями от специфического поведения Fandango. На других площадках различия между критиками и пользователями тоже есть, но это не то же самое, что устойчивое смещение отображаемой оценки вверх.

### 3. `analysis/matching.ipynb`
Главный сравнительный анализ между площадками:

- данные Fandango объединяются с Rotten Tomatoes, Metacritic и IMDb по общим фильмам;
- все оценки приводятся к единой шкале `0..5`;
- строятся распределения оценок для прямого сравнения;
- отдельно анализируются 10 фильмов с худшими оценками Rotten Tomatoes.

Фактические значения на пересечении датасетов:

- общих фильмов: `145`;
- средняя оценка `Fandango STARS`: `4.09`;
- средняя оценка `Fandango RATING`: `3.84`;
- средняя оценка `Rotten Tomatoes critics`: `3.03`;
- средняя оценка `Rotten Tomatoes users`: `3.18`;
- средняя оценка `Metacritic critics`: `2.94`;
- средняя оценка `IMDb`: `3.37`.

Для 10 худших фильмов по Rotten Tomatoes:

- средняя оценка `Fandango STARS`: `3.70`;
- средняя оценка `Fandango RATING`: `3.44`;
- средняя оценка `Rotten Tomatoes critics`: `0.47`;
- средняя оценка `Rotten Tomatoes users`: `1.75`;
- средняя оценка `Metacritic critics`: `1.39`;
- средняя оценка `IMDb`: `2.60`.

Это особенно сильный аргумент: даже у фильмов, которые другие площадки оценивают очень низко, Fandango сохраняет сравнительно высокие оценки.

## Вывод
Гипотеза проекта **подтверждается на используемой выборке**. Анализ показывает два важных эффекта:

1. Внутри самого Fandango отображаемые звезды (`STARS`) в среднем выше численного рейтинга пользователей (`RATING`).
2. При сравнении с Rotten Tomatoes, Metacritic и IMDb оценки Fandango систематически смещены вверх, включая фильмы с очень слабыми внешними оценками.

Корректная формулировка вывода такая: проект показывает **сильные признаки завышения рейтингов на Fandango по сравнению с другими сайтами** и **визуального округления вверх внутри самого Fandango**.
